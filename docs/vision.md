# Vision

## Missao
Oferecer um CMS editorial moderno com experiencia de publicacao consistente, operacao previsivel e base tecnica segura para evolucao.

## Direcao Tecnica
- Runtime de desenvolvimento/testes: SQLite local.
- Runtime de producao: Cloudflare (Worker + D1 + R2 binding).
- Fonte unica de verdade para schemas e tipos compartilhados.
- Server actions centralizadas em `src/server/actions`.
